# event

このリポジトリは「Code for Fukui」によるイベント情報を管理するためのものです。

## デモ
イベント情報は、https://code4fukui.github.io/event/ で公開されています。

## 機能
- イベント情報の管理
- イベントページの自動生成

## データ・API
イベント情報は、Markdown形式のファイルで管理されています。
